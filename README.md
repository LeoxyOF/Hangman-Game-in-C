# Hangman-Game-in-C

--English--

This is my first project, created after 15 days of learning how to code in C.

About the Game:
It's a classic Hangman game. The computer picks a word, and you have to guess it letter by letter before the man is fully drawn.

Lives: 6

Word List: Computer and technology-themed words.

--Português--

Sobre o Jogo:
É um clássico jogo da forca. O computador escolhe uma palavra e você precisa adivinhar letra por letra antes que o boneco seja desenhado por completo.
