# Hangman-Game-in-C

--English--

This is my first project, created after 15 days of learning how to code in C.

About the Game:
It's a classic Hangman game. The computer picks a word, and you have to guess it letter by letter before the man is fully drawn.

Lives: 6

Word List: Initially there are 495 words in the words.txt file, but you can modify it or add any words you desire to play with.

--Português--

Este é meu primeiro projeto, criado após 15 dias aprendendo a programar em C.

Sobre o Jogo:
É um clássico jogo da forca. O computador escolhe uma palavra e você precisa adivinhar letra por letra antes que o boneco seja desenhado por completo.

Vidas: 6

Lista de Palavras: Inicialmente existem 495 palavras no arquivo words.txt, mas você pode modificar ou adicionar palavras que você deseja jogar com.
